Option 1 — Wave (Afrique de l'Ouest)

1. Telechargez l'app Wave sur votre telephone
2. Creez un compte Wave dans votre pays
3. Envoyez directement au numero : +225 07 48 93 56 86
4. Objet : Frais d'activation + votre nom

Disponible : Senegal, Mali, Burkina, Guinee, Cameroun...

Option 2 — WorldRemit (Europe, USA, Canada, Monde)

1. Allez sur worldremit.com ou telechargez l'app WorldRemit
2. Choisissez Cote d'Ivoire comme destination
3. Choisissez Mobile Money — Wave
4. Entrez le numero : +225 07 48 93 56 86
5. Payez en euros, dollars ou votre devise locale

Disponible dans plus de 50 pays

Option 3 — Western Union (Tous pays du monde)

1. Allez dans une agence Western Union de votre ville
2. Dites : Envoyer vers Cote d'Ivoire
3. Nom du beneficiaire : voir avec l'administrateur
4. Telephone : +225 07 48 93 56 86
5. Vous recevrez un code MTCN a noter absolument

Disponible dans presque tous les pays du monde

Option 4 — Wise (Europe principalement)

1. Allez sur wise.com
2. Choisissez Envoyer vers Cote d'Ivoire
3. Choisissez Mobile Money
4. Entrez le numero : +225 07 48 93 56 86

Taux de change excellent, frais reduits
